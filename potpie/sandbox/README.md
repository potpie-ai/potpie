# Potpie Sandbox

Potpie Sandbox provides workspace, runtime, repository, and parser-runner
building blocks for isolated code execution flows used by Potpie.

The Python distribution is published as `potpie-sandbox` and exposes the
`sandbox` import package.

## Installation

```bash
pip install potpie-sandbox
```

For Daytona runtime support:

```bash
pip install "potpie-sandbox[daytona]"
```

## Console Scripts

- `potpie-parse`: parser shim used by local and image-backed sandbox runtimes
