# Potpie Context Engine

Potpie Context Engine provides the context graph API, CLI, MCP server, webhook
handlers, and core application/domain services used by Potpie.

The Python distribution is published as `potpie-context-engine`. Its current
import layout exposes top-level packages such as `domain`, `application`,
`adapters`, and `bootstrap`; keep that in mind when embedding it in larger
Python environments.

## Installation

```bash
pip install "potpie-context-engine[all]"
```

## Console Scripts

- `potpie`: context graph CLI
- `potpie-mcp`: MCP server entry point
